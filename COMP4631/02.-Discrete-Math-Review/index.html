    Discrete Mathematics Review

    Sets

    A set is a collection of distinct objects.

    Members and Elements

    • The objects that make up a set are called members or elements of the set.
    • An object is anything that is “meaningful”. For example: a number, equation, other sets, etc…

    Equality of Sets

    • Two sets are equal if an only if they have the same members. That is, a set is completely determined by its members.
    • This is the principle of extension.

    Set Notation

    A set can be described by listing comma seperated members within curly braces

    Example:

    Let $S = \{1,3,9\}$

    $S$ is a set, and $1,3,9$ are its members.

    It is important to note that order and repetition do not matter in sets. That is, $\{1,3,9\}$ is equivalent to $\{9,1,3\}$ and $\{1,1,1,1,9,9,9,9,3,9,3\}$. They all denote a set whose members are $1, 3, \text{and} \space 9$.

    The Membership Symbol $\in$

    • The fact that $x$ is a member of $S$ can be expressed as $x \in S$.
    • The membership symbol $\in$ can be read as “is in”, “is a member of”, or “belongs to”.
    • The negation of $\in$ is $\not\in$

    Example:

    Let $S = \{7,13,21,47\}$, then $7 \in S, 13 \in S, 21 \in S, 47 \in S, 56 \not\in S$.

    Defining a Set by Membership Properties

    $$S = \{x \in T \space| \space P(x)\}$$ The members of $S$ are members of an already known set $T$ that satisfy property $P$.

    Example:

    • Let $\Bbb Z$ be the set of integers.
    • Let $\Bbb Z_+$ be the set of positive integers.
    • Then $\Bbb Z_+ = \{x \in \Bbb Z \space | \space x > 0\}$

    The Empty Set and Sets of Numbers

    The Empty Set

    • The empty set is also called the null set.
    • It is a set with no members.
    • It is denoted as $\emptyset$. Thus $\{\} = \emptyset$
    • For any object $x, x\not\in \emptyset$.

    The Sets of Positive, Negative, and All Integers

    • $\Bbb Z$ is the set of all integers.
      • $\Bbb Z = \{…, -2, -1, 0, 1, 2, …\}$
    • $\Bbb Z_+$ is the set of all positive integers.
      • $\Bbb Z_+ = \{1,2,3,…\} = \{x\in \Bbb Z \space | \space x > 0\}$
    • $\Bbb Z_-$ is the set of all positive integers.
      • $\Bbb Z_- = \{\space…,-3,-2,-1\} = \{x\in \Bbb Z \space | \space x < 0\}$

    The set of Real Numbers and Rational Numbers

    • $\Bbb R$ is the set of all real numbers.
    • $\Bbb Q$ is the set of all rational numbers.
    • $\Bbb Q = \{x \in \Bbb R \space | \space x = p/q; \space p,q \in \Bbb Z; \space q \not = 0\}$

    Subsets and Power Sets

    Subsets ($\subset, \subseteq$)

    $A$ is a subset of $B$, or $B$ is a superset of $A$ if and only if every member of $A$ is a member of $B$

    Notation:

    $$A \subseteq B \text{ iff } \space \forall x, x \in A \implies x \in B$$

    Example:

    $$\{-2,0,8\} \subseteq \{-3,-2,-1,0,2,4,6,8,10\}$$

    Negation of $\subseteq$

    $A$ is not a subset of $B$, or $B$ is not a superset of $A$ if and only if there is a member of $A$ that is not a member of $B$

    Notation:

    $$A \not \subseteq \text{ iff } \space \exists x, x \in A \text{ and } x \not \in B$$

    Example:

    $$\{2,4\} \not \subseteq \{2,3\}$$

    Obvious Subsets

    • $S \subseteq S$
    • $\emptyset \subseteq S$
      • Vacuously true: $\text{if } x \in \emptyset \implies x \in S$
      • By contradiction: $\text{if } \emptyset \not \subseteq S \implies \exists x, x \in \emptyset \text{ and } x \not \in S$

    Proper Subsets $\subset$

    $A$ is a proper subset of $B$, or $B$ is a proper superset of $A$ if and only if $A$ is a subset of $B$ and $A$ is not equal to $B$.

    Notation:

    $$A \subset B \text{ iff } A \subseteq B \text{ and } A \not = B$$

    Examples:

    $$S \not = \emptyset \implies \emptyset \subset S$$

    $$\Bbb Z_+ \subset \Bbb Z \subset \Bbb Q \subset \Bbb R$$

    Power Sets

    The set of all subsets of a set is called the power set of the set.

    The power set of $S$ is $P(S)$

    Examples:

    • $P(\emptyset) = \{\emptyset\}$
    • $P(\{1,2\}) = \{\emptyset, \{1\}, \{2\}, \{1,2\}\}$
    • $P(S) = \{\emptyset, …, S\}$

    $\in$ and $\subseteq$ are different!

    Examples:

    • $1 \in \{1\}$ is true.
    • $1 \subseteq \{1\}$ is false.
    • $\{1\} \subseteq \{1\}$ is true.

    Questions

    1. Which of the following statements is true?
      • $S \subseteq P(S)$
      • $S \in P(S)$
    2. What is $P(\{1,2,3\})$?

    Equality of Sets by Mutual Inclusion

    Sets $A$ and $B$ have the same members if and only if they mutually include. Meaning, $A \subseteq B \text{ and } B \subseteq A$.

    That is, $A = B \text{ iff } A \subseteq B \text{ and } B \subseteq A$

    Mutual inclusion is very useful for proving the equality of two sets. To prove and equality, we prove two subset relationships.

    Example Showing the Equality of Sets:

    • Recall that $\Bbb Z$ is the set of all integers.
    • Let $A = \{x \in \Bbb Z \space | \space x = 2m \space \text{for some } m \in \Bbb Z \}$
    • Let $B = \{y \in \Bbb Z \space | \space y = 2n-2 \space \text{for some } n \in \Bbb Z\}$
    • To show that $A \subseteq B$, note that
      • $2m = 2(m+1)-2 = 2n-2$
    • To show that $B \subseteq A$, note that
      • $2n-2 = 2(n-1) = 2m$
    • That is, $A=B$
    • Both $A$ and $B$ denote the set of even integers.

    Exercise:

    • Let
      • $A = \{x \in \Bbb Z \space | \space x^2 -1 = 0\}$
      • $B = \{x \in \Bbb Z \space | \space 2x^3 - x^2 -2x +1 = 0\}$

    Show that $A = B$ by the method of mutual inclusion.

    Universal Sets, Venn Diagrams

    Universal Sets

    • Depending on the context of discussion:
      • Define a set $U$ such that all sets of interest are subsets of $\Bbb U$.
      • The set $\Bbb U$ is known as a universal set.

    Example:

    • When dealing with integers, $\Bbb U$ may be $\Bbb Z$.
    • When dealing with plane geometry, $\Bbb U$ may be the set of all points in the plane.

    Venn Diagrams

    • Used to visualize relationships among some sets
    • Each subset of $\Bbb U$ is represented by a circle/oval inside of a rectangle.

    Set Operations

    Let $A, B$ be subsets of some universal set $\Bbb U$.
    The following set operations create new sets from $A$ and $B$:

    • Union
      • $A \cup B = \{x \in \Bbb U \space | \space x \in A \text{ or } x \in B\}$
      • Example: $$\{1,2,3\} \cup \{2,3,4,5\} = \{1,2,3,4,5\}$$
    • Intersection
      • $A \cap B = \{x \in \Bbb U \space | \space x \in A \text{ and } x \in B\}$
      • Example: $$\{1,2,3\} \cap \{2,3,4,5\} = \{2,3\}$$
    • Difference
      • $A - B = A \backslash B = \{x \in \Bbb U \space | \space x \in A \text{ and } x \not \in B\}$
      • Example: $$\{1,2,3\} - \{2,3,4,5\} = \{1\}$$
    • Complement
      • $A^c = \Bbb U - A = \{x \in \Bbb U \space | \space x \not \in A\}$

    Set Identities

    • Commutative Laws:
      • $A \cup B = B \cup A$
      • $A \cap B = B \cap A$
    • Associative Laws:
      • $(A \cup B) \cup C = A \cup (B \cup C)$
      • $(A \cap B) \cap C = A \cap (B \cap C)$
    • Distributive Laws:
      • $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
      • $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
    • $\emptyset$ is the identity for union:
      • $\emptyset \cup A = A \cup \emptyset = A$
    • $\Bbb U$ is the identitiy for intersection:
      • $A \cap \Bbb U = \Bbb U \cap A = A$
    • Double Comlement Law:
      • $(A^c)^c = A$
    • Idempotent Laws:
      • $A \cup A = A$
      • $A \cap A = A$
    • De Morgan’s Laws:
      • $(A \cup B)^c = A^c \cap B^c$
      • $(A \cap B)^c = A^c \cup B^c$

    Questions

    1. What is $(A \cap B) \cap (A \cup B)$?
    2. What is $(A \cup B) \cup (A \cap B)$?
