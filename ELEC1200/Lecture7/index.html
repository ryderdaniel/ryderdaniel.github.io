    Motivation: Equalization

    • The channel introduces ISI which causes the eye to close.
    • The goal of a channel equalizer is to undo the effect of the channel.
    • This will cause the eye to open.

    Developing the Equalizer

    • How can we figure out what to put into the channel equalizer?
      • We need to describe the effect of the channel using a model that enables us to predict the channel output for any input.
      • Using this model, we can hopefully guess (estimate) what the input to the channel was from knowledge of the output.

    Equivalent Models

    Model 1

    • We have seen that the communication channel can be modelled by:

      • Assuming it is linear and time invariant
      • Assuming it has a known step response $s(n)$

    • For our communication system, the step response has the form
      $s(n) = k(1-a^{n+1})u(n),\ 0 \leq a \leq 1$

    • This model is useful in predicting the output for a given input. However, it is not easy to go the other way.

    • We seen another equivalent model that will allow us to do this easily.

    Equivalent Model

    • A model of the channel is a way of predicting the channel output given the channel input.

    • One system may be described by many equivalent models that are written in many different ways.

    • Given the same input, Equivalent models make the same prediction for the output.

    • Why do we need more than one model?

      • The more models we have, the better we can understand the system
      • Different models have different uses.

    Recursive Channel Models

    • Recursive = involving repeated application of a rule

    • A recursive model for a discrete time waveform $x(n)$ has two parts:

      • A formula that defines the nth sample in terms of past samples
      • An initial starting condition

    • Generating the waveform by recursion:

      • Given $x(0)$, find $x(1) = f(x(0))$
      • Given $x(1)$, find $x(2) = f(x(1))$
      • $…$

    • It turns out that the response of the channel $y(n)$ to an input $x(n)$ can be described by a recursive formula:
      $y(n) = a \cdot y(n-1) + (1-a) \cdot k \cdot x(n)$

    • The parameter $a$ lies between $0$ and $1$
    • The parameter $k$ scales the input
    • This is also known as a feedback system. Because the output feeds back as input to the system to determine the next output.

    Effect of Parameter $a$

    The parameter determines the memory in the channel

    • $a=0$:
      • no memory of the past
      • $y(n) = k \cdot x(n)$
      • The channel output is just the input multiplied by $k$
    • $a = 1$
      • Infinite memory of the past
      • $y(n) = y(n-1)$
      • Channel output is constant, ignores the channel input.

    Proof of Equivalence

    • The two channel models are equivalent
      • Model 1: The channel is an LTI system with step response
        $s(n) = k(1-a^{n+1})u(n),\ 0 \leq a \leq 1$
      • Model 2: The channel has input $x(n)$ and output $y(n)$ determined by
        $y(n) = a \cdot y(n-1) + (1-a) \cdot k \cdot x(n), \ 0 \leq a \leq 1$
    • We can establish equivalence by showing that:
      • Model 2 is LTI.
      • Model 2 has a step resoponse to a unit step given by $s(n)$

    Fact 1: Model 2 is LTI

    $$y(n) = a \cdot y(n-1) + (1-a) \cdot k \cdot x(n)$$

    • It is linear:
      • Intuitively, the output is a linear function of the input and past outputs.
      • Formally, we will prove that it satisfies the two properties:
        • Homogeniety:
          • $y(n) = a \cdot y(n-1) + (1-a) \cdot k \cdot x(n)$
          • $c\cdot y(n) = c\cdot (a \cdot y(n-1) + (1-a) \cdot k \cdot x(n))$
          • $c\cdot y(n) = c \cdot a \cdot y(n-1) + (1-a) \cdot k \cdot c \cdot x(n)$
          • $\therefore c\cdot x(n) \to |\text{system}| \to c \cdot y(n)$
        • Additivity:
          • Add two of the equations together and see
    • It it time invariant
      • Intuitively, the parameters $k$ and $a$ do not change with $n$.
      • Formally we prove that if we delay the input by $d$, the output is the same just delayed by $d$.
        • substitute $n$ with $n-d$. Observe that $x(n) \to x(n-d)$
